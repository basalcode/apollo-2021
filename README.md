다운로드 및 패키지 설치
---
* ### 다운로드
```PowerShell
git clone https://github.com/basalcode/apollo-2021.git
```
* ### 패키지 설치
```PowerShell
npm install

또는

yarn
```
시작하기
---
```PowerShell
npm run start

또는

yarn start
```
localhost:3000으로 실행합니다.<br />
